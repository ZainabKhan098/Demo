History of HTML:

In 1980, physicist Tim Berners-Lee, a contractor at CERN proposed and prototyped ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee wrote a memo proposing an Internet-based hypertext system. Berners-Lee specified HTML and wrote the browser and server software in late 1990. That year, Berners-Lee and CERN data systems engineer Robert Cailliau collaborated on a joint request for funding, but the project was not formally adopted by CERN. In his personal notes of 1990, Berners-Lee listed "some of the many areas in which hypertext is used" an encyclopedia is the first entry.

What is HTML

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript, a programming language.

HTML uses a system of tags and attributes to organize and format text, images, links, and other elements on a webpage. These tags, enclosed in angle brackets (e.g., <p> for paragraphs or <h1> for headings), tell the browser how to render the content. Without HTML, webpages would lack structure and consistency, making it difficult to navigate or style them effectively.

Over the years, HTML has evolved through multiple versions, with HTML5 being the latest major iteration. HTML5 introduced new semantic elements (like <header>, <article>, and <footer>), multimedia support (e.g., <video> and <audio> tags), and improved APIs for richer web applications. Today, HTML remains the foundation of the modern web, working alongside CSS for styling and JavaScript for interactivity to create dynamic, user-friendly experiences.

Versions of HTML

  1. HTML 1.0 (1993)
    • First version with basic tags (headings, paragraphs, lists).
  2. HTML 2.0 (1995)
    • Standardized features; added forms.
  3. HTML 3.2 (1997)
    • Added tables, inline styles, and scripting.
  4. HTML 4.0 (1997)
    • CSS support, better structure, and scripting improvements.
  5. HTML5 (2014)
    • Semantic tags, multimedia (<video>), APIs, mobile-friendly.
